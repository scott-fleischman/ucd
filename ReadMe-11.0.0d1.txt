This directory contains PRELIMINARY data files in preparation for beta review
for Version 11.0.0 of the Unicode Standard.

The "ucd" subdirectory contains the Unicode
Character Database data files.

The "charts" subdirectory WILL CONTAIN an archival set of
pdf code charts corresponding exactly to Version 11.0.0.

The "ucdxml" subdirectory WILL CONTAIN the XML version of
the Unicode Character Database.

Zipped versions of UCD data files for Version 11.0.0 WILL BE 
posted in:

http://www.unicode.org/Public/zipped/11.0.0/
